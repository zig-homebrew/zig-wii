# zig-wii

![](screenshot.webp)

## Getting started

- [zig](https://ziglang.org/download/)
- [devkitPro](https://devkitpro.org/wiki/Getting_Started)

```
pacman -S wii-dev
git clone https://github.com/zig-homebrew/zig-wii
cd zig-wii/
zig build # then run zig-out/zig-wii.dol with dolphin-emu
```
